Help
🕵️ Mode A — Account Monitor
1. Select Mode A on the Home screen
2. Tap "Setup Account Monitor" in the bot
3. Get API ID & Hash from my.telegram.org/apps
4. Enter API ID → Hash → Phone → OTP
5. Tap Start Monitoring — done!

You'll receive alerts for DMs, mentions & replies.
📲 Mode B — WA Forwarder
1. Select Mode B on the Home screen
2. Connect WhatsApp via QR code
3. Tap Start Forwarding
4. Anyone who messages this bot → your WhatsApp!

No Telegram login needed. 100% safe.
📱 Connect WhatsApp
Open WhatsApp → Settings → Linked Devices → Link a Device → Scan the QR code that appears in the bot.
❓ FAQ
Q: Is Mode A safe?
A: Yes. Session is stored only on your own server.

Q: Can I switch modes?
A: Yes! Tap "Switch Mode" anytime.

Q: Why no alerts?
A: Check if monitoring/forwarding is active (🟢) in Dashboard.
