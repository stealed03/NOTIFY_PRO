❓ Help
🕵️ Mode A — Account Monitor Setup
1. Home screen pe Mode A select karo
2. Bot mein "Login Telegram Account" tap karo
3. my.telegram.org/apps se API ID & Hash lo
4. API ID → Hash → Phone Number → OTP enter karo
5. Start Monitoring tap karo — ho gaya! ✅

Ab DMs, mentions & replies ka alert milega.
📲 Mode B — WA Forwarder Setup
1. Home screen pe Mode B select karo
2. @BotFather se naya bot banao → token copy karo
3. Bot mein "Set Bot Token" tap karke token paste karo
4. Apna WhatsApp number enter karo
5. Pehle apne WA se admin ke WA pe ek "Hi" bhejo (opt-in)
6. Start Forwarding tap karo — shuru! 🚀

✅ Koi Telegram login nahi chahiye. 100% safe.
📱 WhatsApp Connect Kaise Kare
WhatsApp kholo → Settings → Linked Devices → Link a Device → Bot mein aaya QR code scan karo.
